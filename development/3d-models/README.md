# 3D-Modelle

Die eigentlichen GLB-Modelle (Blätter, Blüten, Kugeln) sind je 54-57MB gross
und werden NICHT im Repo gespeichert, um das Repo klein zu halten.

## Woher

Alle Modelle wurden während dieser Session mit **Tripo3D** aus Referenz-Bildern
generiert. Die Original-GLBs liegen in der Sandbox unter
`/home/user/workspace/glb-output/`:

- `blatt_1.glb` — Grosses Lotus-Blatt (56MB)
- `blatt_2.glb` — Zweites Lotus-Blatt (57MB)
- `bluete_lotus.glb` — Lotus-Blüte (57MB)
- `bluete_rosa.glb` — Rosa Seerose (57MB)
- `bluete_weiss.glb` — Weisse Seerose (57MB)
- `kugel.glb` — Glaskugel v1 (55MB)
- `kugel-v2.glb` — Glaskugel v2 (56MB)

## Preview

Preview-Renderings dieser Modelle (Front, Side, Perspektive) liegen unter
`preview-renders/` als PNGs.

## Wiederherstellung

Falls die GLBs erneut gebraucht werden: entweder frisch via Tripo3D generieren
(Prompt-Historie in `../contactsheets/`) oder in Blender aus den Referenz-Bildern
neu modellieren.
